![Power BI](https://img.shields.io/badge/Tool-PowerBI-yellow?style=for-the-badge&logo=powerbi)
![DAX](https://img.shields.io/badge/Language-DAX-blue?style=for-the-badge)
![Status](https://img.shields.io/badge/Status-Completed-brightgreen?style=for-the-badge)

Dashboard interactivo desarrollado en **Power BI** orientado al análisis de performance comercial y soporte en la toma de decisiones estratégicas.

---

## 🚀 Objetivo del proyecto

Transformar datos transaccionales en **insights accionables**, permitiendo:

- Evaluar el rendimiento de ventas en el tiempo  
- Identificar patrones de comportamiento de clientes  
- Analizar desempeño por producto, ciudad y vendedor  
- Facilitar la toma de decisiones basada en datos  

---

## 🧭 How to use

1. Navegar entre páginas desde el menú inferior  
2. Aplicar filtros por:
   - Ciudad  
   - Año  
   - Método de pago  
3. Analizar:
   - KPIs globales  
   - Evolución de ventas  
   - Segmentación de clientes  
   - Performance de productos y vendedores  

---

## 🧠 Modelo de datos

Se diseñó un modelo relacional tipo **estrella (Star Schema)**.

### 🔹 Tablas:

- **Facturas**
- **DetalleFacturas**
- **Productos**
- **Clientes**
- **Sucursales**
- **Ciudades**
- **Vendedores**
- **Calendario**

✔ Integridad referencial garantizada  
✔ Optimización para análisis temporal  

---

## 📈 KPIs principales

- 💰 Ventas Totales  
- 🧾 Ticket Promedio  
- 📦 Productos Vendidos  
- 📊 Variación % de ventas  
- 📉 Evolución temporal  

---

## 🔐 Seguridad (RLS)

- **Nacional** → acceso total  
- **Regional** → acceso por ciudad  
- **Sucursal** → acceso por sucursal
